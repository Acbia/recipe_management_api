# Recipe Management API

A backend RESTful API built with Django and Django REST Framework that allows users to create, manage, and explore recipes.

## Features
- User authentication
- CRUD operations for recipes
- Categories and ingredients management
- Filter recipes by category or ingredient

## Tech Stack
- Django
- Django REST Framework
- SQLite (development)

## Project Status
This project is currently under active development as part of a backend capstone project.

## Setup Instructions
1. Clone the repository.
2. Create and activate a virtual environment.
3. Install dependencies with `pip install -r requirements.txt`.
4. Run migrations with `python manage.py migrate`.
5. Start the server with `python manage.py runserver`.
